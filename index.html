Webapps

Live on GitHub Pages

MorTweetSRS

SRS X / Twitter

Spaced repetition for bookmarked posts. Recall a cover first, reveal the tweet, grade Again / Hard / Good / Easy. Deck library + local backups.

Open app Source

MorAlgorithms

Feed Ranking

Weight-based multi-platform feed engine. Add items, like or dislike, and let feedback retune what surfaces next. Export taste as JSON or .mora.

Open app Source

More tools ship as separate repos under @MoribundMurdoch and land here when they need a browser front door. Desktop / CLI work (MorFlash, morconcat, etc.) stays on GitHub Releases — not forced into Pages.

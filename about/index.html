Lore · about this site

Hello. I’m MoribundMurdoch: lexicographer, open-source human, recovering fubarist, and someone trying to accelerate the velocity of ideas without stuffing everything into one soulless CMS.

This repository is not the blog. It used to wear a Hugo skin with empty post lists and placeholder widgets. That fought the actual use of the domain: hosting small browser tools next to their source repos.

What lives here now:

  • An inventory of GitHub Pages webapps (MorTweetSRS, MorAlgorithms, …)
  • A toolkit of personal CLIs, desktop tools, themes, and extensions — grouped by job (study, dev, media, themes)
  • An Institute armory of Moribund Institute products (learning systems, theme forge, media studio)
  • Stacks: sister orgs shelved under Dewey main classes — 000 information, 300 social sciences, 400 language — with a compact call table as index
  • A world map of which surface does which job (identity, writing, and the same Dewey zones)
  • Static HTML only: no Hugo, no build step for the hub itself

For essays and neologisms, go to Blogspot. For meta-writing about the multi-site setup, see Bearblog. For the main identity hub, moribundmurdoch.com. For playful web aesthetics, Neocities. For the Institute he founded, moribund.institute. For wearable dictionary craft, WearYourDictionary. For definitions and neologisms, MorDictionary. For flashcard engines, MorFlashcards. For language outposts, MorEnglish (EnglishMoribund, DDC 420) and MandarinMurdoch. For topical shelves, TechMoribundMurdoch and MammonistMurdo. For the classified map of it all, see Stacks and the call table.

About this site

Hello. I’m MoribundMurdoch: lexicographer, open-source human, recovering fubarist, and someone trying to accelerate the velocity of ideas without stuffing everything into one soulless CMS.

This repository is not the blog. It used to wear a Hugo skin with empty post lists and placeholder widgets. That fought the actual use of the domain: hosting small browser tools next to their source repos.

What lives here now:

  • Live apps on GitHub Pages (MorTweetSRS, MorAlgorithms, …)
  • Personal tools: CLIs, desktop apps, themes, and extensions — grouped by what they do
  • Institute projects: Moribund Institute products (learning, themes, media)
  • Projects: related hubs grouped by topic — 000 info & computing, 300 society & learning, 400 language — plus a topic index
  • Other sites: which surface does which job (identity, writing, and the same topic groups)
  • Static HTML only: no Hugo, no build step for the hub itself

For essays and neologisms, go to Blogspot. For meta-writing about the multi-site setup, see Bearblog. For the main identity hub, moribundmurdoch.com. For playful web aesthetics, Neocities. For the Institute he founded, moribund.institute. For wearable dictionary craft, WearYourDictionary. For definitions and neologisms, MorDictionary. For flashcard engines, MorFlashcards. For language outposts, MorEnglish (EnglishMoribund, topic 420) and MandarinMurdoch. For topical shelves, TechMoribundMurdoch and MammonistMurdo. For the full project map, see Projects and the topic index.

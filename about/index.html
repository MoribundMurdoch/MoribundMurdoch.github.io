About this site

Hello. I am MoribundMurdoch: an open-source developer and dictionary writer, and founder of the Moribund Institute. I build tools and writing projects without forcing everything into one heavy CMS.

This repository is not the blog. It used to be a Hugo site with empty post lists and placeholder widgets. That did not match how the domain is used: hosting small browser tools next to their source repos.

What is here now:

  • Live apps on GitHub Pages (MorTweetSRS, MorAlgorithms, and similar)
  • Personal tools: command-line tools, desktop apps, themes, and extensions, grouped by what they do
  • Institute projects: Moribund Institute products for learning, themes, and media
  • Projects: related accounts grouped by topic (000 info and computing, 300 society and learning, 400 language), plus a topic index
  • Other sites: which site or account does which job (identity, writing, and the same topic groups)
  • Static HTML only: no Hugo, no build step for this site itself

For essays and new words, go to Blogspot. For notes about the multi-site setup, see Bearblog. For the main personal site, see moribundmurdoch.com. For experimental web pages, see Neocities. For the Institute, see moribund.institute. For wearable dictionary projects, see WearYourDictionary. For definitions and new words, see MorDictionary. For flashcard engines, see MorFlashcards. For language projects, see MorEnglish (EnglishMoribund, topic 420) and MandarinMurdoch. For other topic groups, see TechMoribundMurdoch and MammonistMurdo. For the full project list, see Projects and the topic index.

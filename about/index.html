Lore · about this site

Hello. I’m MoribundMurdoch: lexicographer, open-source human, recovering fubarist, and someone trying to accelerate the velocity of ideas without stuffing everything into one soulless CMS.

This repository is not the blog. It used to wear a Hugo skin with empty post lists and placeholder widgets. That fought the actual use of the domain: hosting small browser tools next to their source repos.

What lives here now:

  • An inventory of GitHub Pages webapps (MorTweetSRS, MorAlgorithms, …)
  • A field kit of personal CLIs, desktop tools, themes, and extensions
  • An Institute armory of Moribund Institute products (MorFlash, theme editor, LMS vault, …)
  • Constellations: classification of WearYourDictionary, MorDictionary, MorFlashcards, Institute, and personal tools
  • A world map of which surface does which job
  • Static HTML only: no Hugo, no build step for the hub itself

For essays and neologisms, go to Blogspot. For meta-writing about the multi-site setup, see Bearblog. For the main identity hub, moribundmurdoch.com. For playful web aesthetics, Neocities. For the Institute he founded, moribund.institute. For wearable dictionary craft, WearYourDictionary. For definitions and neologisms, MorDictionary. For flashcard engines, MorFlashcards.

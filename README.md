## Tech stack

Plain static HTML, CSS, and a tiny `js/site.js`. **No Hugo.** Edit files, push to `main`, Pages serves them.

```
index.html       # live apps, personal tools, institute, projects, other sites
about/           # what this site is for
css/site.css
js/site.js
img/mor-icon.png
robots.txt
sitemap.xml
```
